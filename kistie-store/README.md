# Kistie Store

A modern, full-stack fashion e-commerce demo project: inventory management, admin dashboard, shopping cart, and more. Built with Node.js/Express, SQLite, and a responsive HTML/CSS/JavaScript frontend. Deployed live for easy review.

## Project Overview
Kistie Store is a professional e-commerce platform for fashion retail, featuring:
- Admin authentication and dashboard
- Product catalog with images, prices, and details
- Add, edit, and delete products (CRUD)
- Shopping cart and order placement
- RESTful API (Node.js/Express)
- SQLite database for persistent storage
- Responsive, modern UI (HTML, CSS, JS)
- CORS and security best practices

## Tech Stack
- Backend: Node.js, Express, SQLite
- Frontend: HTML, CSS, JavaScript
- Deployment: Render.com

## Getting Started

### Local Setup
1. Clone the repo:
   ```
   git clone https://github.com/dallas8000-ops/JNalumansi.git
   cd JNalumansi/kistie-store/backend
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the backend server:
   ```
   npm start
   ```
4. Open `kistie-store/index.html` in your browser for the frontend.

### Admin Login
- Default admin credentials are set up via the backend. (See backend/README.md for setup.)
